## 📸 Cara Menampilkan Gambar

Setiap project memiliki folder `images/` untuk menyimpan screenshot dan aset visual. Gunakan format Markdown berikut untuk menampilkan gambar di README atau FEATURES.md:

### Format Dasar:
```markdown
![Deskripsi Gambar](./images/nama-gambar.png)
```

### Contoh dengan Link:
```markdown
[![Klik untuk melihat detail](./images/screenshot.png)](./images/screenshot.png)
```

### Dengan Ukuran:
```markdown
<img src="./images/screenshot.png" width="600" alt="Deskripsi">
```
