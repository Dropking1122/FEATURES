## 🎯 Tentang Project

TELEBOT-ORDER adalah bot Telegram yang dirancang untuk jualan produk digital secara otomatis. Pelanggan cukup chat bot, pilih produk, bayar, dan produk langsung dikirim — tanpa perlu admin standby 24 jam.

Bot ini cocok untuk:
- Penjual akun premium (Netflix, Spotify, Canva, dll)
- Reseller produk digital
- Toko online berbasis Telegram

---

## 🎯 Fitur Utama

✅ **Katalog Produk Otomatis**
- Tampilkan daftar produk lengkap dengan harga
- Kategori produk yang terorganisir
- Gambar produk langsung di chat

✅ **Pemesanan Mudah**
- Pelanggan pilih produk dan langsung checkout
- Proses pemesanan step-by-step yang mudah
- Konfirmasi pesanan otomatis

✅ **Multi Metode Pembayaran**
- QRIS otomatis
- Transfer bank manual dengan konfirmasi admin
- Saldo akun pelanggan

✅ **Pengiriman Produk Digital Otomatis**
- Key/akun digital dikirim otomatis setelah bayar
- Stok berkurang otomatis
- Riwayat transaksi tersimpan

✅ **Panel Admin**
- Kelola produk dan stok via command Telegram
- Lihat laporan penjualan
- Notifikasi pesanan masuk real-time

## 🛠️ Teknologi

- **Platform:** Telegram Bot API
- **Backend:** Node.js / Python
- **Database:** PostgreSQL / SQLite
- **Payment:** QRIS / Manual Transfer
