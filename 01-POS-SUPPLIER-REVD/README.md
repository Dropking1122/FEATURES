# 1️⃣ POS-SUPPLIER-REVD

**Sistem Point of Sale (POS) untuk Supplier**

Aplikasi POS lengkap dengan manajemen stok, transaksi, hutang pelanggan, dan laporan keuangan untuk supplier/toko retail.

---

## 🎯 Fitur Utama

✅ **Autentikasi & Manajemen Akun**
- Login dengan email/password  
- Sistem 2 level akses (Admin & Kasir)
- Reset password via email

✅ **Manajemen Produk & Stok**
- CRUD Produk dengan validasi harga
- Stok tracking dengan alert minimum
- Import/Export produk via Excel/CSV

✅ **Point of Sale (POS)**
- Keranjang belanja dengan multi-item
- Pilih tipe harga (Grosir/Ecer) per item
- Mode input qty atau sisa stok
- Warning otomatis jika rugi

✅ **Hutang & Transaksi Pelanggan**
- Manajemen hutang pelanggan
- Cicilan otomatis
- Riwayat transaksi per pelanggan

✅ **Laporan & Analytics**
- Dashboard dengan grafik penjualan
- Laporan profit harian/mingguan/bulanan
- Statistik produk terlaris
- Laporan stok menipis

✅ **Admin Panel**
- Interface user-friendly
- Mobile-responsive design
- Export laporan ke PDF/Excel

## 🛠️ Teknologi

- **Backend:** Laravel
- **Frontend:** Blade Template + Alpine.js
- **Database:** MySQL
- **Authentication:** Laravel Breeze
