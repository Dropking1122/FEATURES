# Catatan Lengkap Fitur — Website CLC (Computer Learning Center)

> Website resmi lembaga kursus komputer CLC — platform informasi, pendaftaran online, dan manajemen kelas.

🔗 **Live:** [clccourse.online](https://clccourse.online)

---

## 1. Landing Page (Halaman Utama)

### Hero Section
- Tampilan modern dengan animasi
- Tagline dan deskripsi singkat lembaga
- Tombol CTA "Daftar Sekarang" yang mencolok
- Responsif di semua ukuran layar

### Navigasi
- Menu navbar sticky (mengikuti scroll)
- Navigasi smooth-scroll ke setiap section
- Mobile-friendly dengan hamburger menu

---

## 2. Informasi Program Kursus

### Daftar Program
| Program | Keterangan |
|---------|------------|
| Microsoft Office | Word, Excel, PowerPoint — dari dasar hingga mahir |
| Desain Grafis | Photoshop, CorelDraw, Canva |
| Pemrograman Dasar | HTML, CSS, JavaScript untuk pemula |
| Akuntansi Komputer | MYOB, Zahir, laporan keuangan digital |
| Mengetik 10 Jari | Teknik mengetik cepat dan akurat |

### Detail Per Program
- Kurikulum dan materi yang diajarkan
- Durasi kursus (jumlah pertemuan / jam)
- Biaya pendaftaran dan SPP
- Sertifikat yang didapat setelah lulus
- Jadwal kelas yang tersedia

---

## 3. Pendaftaran Online

### Form Pendaftaran
- Nama lengkap calon siswa
- Nomor HP / WhatsApp aktif
- Program kursus yang diminati
- Pilihan jadwal (pagi / siang / sore / malam)
- Asal sekolah / instansi (opsional)

### Proses Setelah Daftar
- Konfirmasi otomatis dikirim ke WhatsApp pendaftar
- Admin mendapat notifikasi pendaftaran baru
- Tim CLC menghubungi pendaftar untuk konfirmasi jadwal
- Data tersimpan di database untuk follow-up

---

## 4. Dashboard Admin

### Manajemen Pendaftar
- Lihat semua data pendaftar masuk
- Filter berdasarkan program dan tanggal
- Update status: Baru, Dikonfirmasi, Aktif, Selesai
- Export data pendaftar ke Excel

### Manajemen Konten
- Edit teks landing page (hero, about, program)
- Upload dan kelola galeri foto
- Tambah / edit / hapus testimoni
- Kelola informasi jadwal dan biaya kursus

### Statistik
- Total pendaftar per bulan
- Program kursus terpopuler
- Grafik pendaftaran bulanan
